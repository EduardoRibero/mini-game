# Jogo de Adivinhação de Palavras em React

## Descrição
Bem-vindo ao Jogo de Adivinhação de Palavras em React! Este projeto é uma aplicação simples em React onde os jogadores podem tentar adivinhar palavras, receber dicas, acumular pontuações e reiniciar o jogo.

## Funcionalidades
- Adivinhe a palavra correta para ganhar pontos.
- Receba dicas para ajudar na adivinhação.
- Limite de erros para aumentar o desafio.
- Pontuação final exibida ao término do jogo.
- Opção para reiniciar o jogo.

## Como Jogar
1. Clone o repositório para o seu ambiente local.
2. npm install ou yarn install
3. npm start  ou yarn start
